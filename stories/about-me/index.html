About me

Probability, computation and programming are what I like.

These are the tools used for my math problems:

  • Sagemath, the best free Computer Algebra System out there.
  • J, an array programming language.
  • GNU Compiler Collection, available by default in most linux distros.
  • Experimental mathematician's toolkit by David H. Bailey and others.
  • FriCAS, another CAS, which has some very good sequence guessing routines missing in Sage.
